App Starter
===========

Modern application starter kit:

* [HTML5 Boilerplate][1]
* [AngularJS][2]
* [Express][3]
* [Grunt][4]
* [LiveReload][5]


Requirements
------------

    $ npm install -g grunt
    $ npm install -g bower


Installation
------------

    $ git clone git@github.com:ericclemmons/app-starter.git
    $ cd app-starter
    $ npm install


Usage
-----

Start an [Express][3] server with [LiveReload][5]:

    $ grunt server


Customize
---------

- `public/crossdomain.xml`
- `public/humans.txt`
